# MapMoPad
### 내 머릿속 지도 만들기, 맵모장 🗺️
----

### 맵모장의 특징
#### ✔️ 내용만 입력하면 자동으로 분류해줍니다.
#### ✔️ 카테고리가 나오지 않아도 당황하지마세요!
##### (메모 5개를 만들어보세요, 그래도 안되면 키워드를 찾는중입니다.)
#### ✔️ 추출된 키워드를 조회할 수 있고, 키워드를 갖는 메모를 분류해줍니다.
